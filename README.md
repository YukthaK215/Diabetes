# 🩺 Diabetes Prediction Using Machine Learning

## 📌 Project Overview

The **Diabetes Prediction System** is a Machine Learning project developed using Python and Google Colab. The project predicts whether a person is likely to have diabetes based on various health-related parameters. It demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, and prediction.

This project showcases the practical application of machine learning in healthcare for early diabetes risk assessment.

---

## 🎯 Objectives

- Analyze diabetes-related healthcare data.
- Perform data cleaning and preprocessing.
- Train a machine learning classification model.
- Predict diabetes based on patient health metrics.
- Evaluate model performance using standard metrics.

---

## 📊 Dataset

The project uses the **Pima Indians Diabetes Dataset** from Kaggle.

### Features

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome

## 🚀 How to Run the Project

### Clone the Repository

```bash
git clone https://github.com/YukthaK215/Diabetes-Prediction.git
```

### Navigate to the Project Directory

```bash
cd Diabetes-Prediction
```

### Install Dependencies

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

### Run the Notebook

Open the notebook in Google Colab or Jupyter Notebook and execute all cells.
